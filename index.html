Investigación
El primer paso es hacer una investigación. Conoceré todo lo posible sobre su proyecto para tener una buena estimación de la envergadura del mismo..
Oferta personalizada
Después de obtener toda la información haré una lista con todos los pasos necesarios para hacer realidad su proyecto. Con esto viene el precio fijo y el calendario.
Desarrollo del proyecto
A partir de aquí, el proyecto comienza y yo haré todo el trabajo en esta fase. Dependiendo del tamaño del proyecto, crearé una página web personalizada (Trello) donde podrás seguir todo el progreso.
Rondas de retroalimentación
En el momento de la entrega, recibirás el proyecto y el código fuente (cuando se haya discutido). La retroalimentación puede continuar indefinidamente siempre que esté dentro del alcance del proyecto.
